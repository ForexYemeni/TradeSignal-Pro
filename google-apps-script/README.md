# TradeSignal Pro — نظام تداول ذكي

## 📋 ملخص المشروع

نظام احترافي متكامل لاستقبال إشارات TradingView وتحليلها وتحويلها إلى بيانات منظمة. يتكون من:

1. **Google Apps Script** — محلل نص ذكي يعمل كسيرفر خلفي
2. **تطبيق ويب Next.js** — واجهة مستخدم + لوحة أدمن

---

## 🚀 التركيب والاستخدام

### Google Apps Script

1. افتح [script.google.com](https://script.google.com)
2. أنشئ مشروع جديد
3. انسخ محتوى الملف `google-apps-script/TradeSignal-Parser.js`
4. من القائمة: **Deploy > New deployment**
5. اختر **Web app** وحدد Execute as: **Me**
6. انسخ رابط Webhook واستخدمه في TradingView

### تطبيق الويب

1. ثبّت التبعيات: `bun install`
2. شغّل قاعدة البيانات: `bun run db:push`
3. شغّل الخادم: `bun run dev`

---

## 📡 طريقة إرسال الإشارات

### من TradingView

في TradingView، أضف Webhook URL إلى تنبيهك:

```
https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec
```

مع Body:
```
text=🟢 إشارة شراء 🚀
📌 XAUUSD │ 15 │ 1س
⭐ ⭐⭐

🔵 الدخول: 2350.5
🔴 الوقف : 2345.0

🎯 TP1: 2355
🎯 TP2: 2360
🎯 TP3: 2370
```

### عبر API (للاختبار)

```bash
curl -X POST http://localhost:3000/api/signals \
  -H "Content-Type: application/json" \
  -d '{"text": "🟢 إشارة شراء 🚀\n📌 XAUUSD │ 15 │ 1س\n⭐ ⭐⭐\n\n🔵 الدخول: 2350.5\n🔴 الوقف : 2345.0\n\n🎯 TP1: 2355\n🎯 TP2: 2360\n🎯 TP3: 2370"}'
```

## 🧠 المحلل (Parser)

يدعم المحلل:

| الحقل | الأنماط المدعومة |
|-------|-----------------|
| النوع | 🟢 شراء / BUY / Long / 🔴 بيع / SELL / Short |
| الزوج | XAUUSD, EUR/USD, BTC/USDT, ... |
| الدخول | الدخول: 2350.5 / Entry: 2350.5 |
| الوقف | الوقف: 2345.0 / SL: 2345.0 / Stop Loss: 2345.0 |
| الأهداف | TP1: 2355 / الهدف 1: 2355 / Take Profit 1: 2355 |
| الثقة | ⭐⭐⭐ (عدد النجوم) |
| الإطار | 1س, 5س, 15س, 1H, 4H, يومي |

---

## 📱 واجهة المستخدم

- عرض الإشارات كـ Cards احترافية
- تلوين أخضر = شراء / أحمر = بيع
- عرض النجوم ومستوى الثقة
- عرض الدخول / SL / TP بشكل منظم
- فلترة حسب الحالة

## 🛠 لوحة الأدمن

- إحصائيات شاملة (نسبة فوز، أزواج، ...)
- تعديل حالة الإشارة (وصل الهدف، وصل الوقف، ...)
- حذف الإشارات
- فلترة متقدمة
